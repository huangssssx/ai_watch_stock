# 尾盘选股策略 - 更新说明

## 📝 版本更新记录

### v0.2.0 (2024-01-XX) - Tushare 接口临时替代版

**更新原因**：
- Tushare 接口存在访问限制和稳定性问题
- 为保证策略可用性，暂时屏蔽 Tushare 相关功能

**主要变更**：

#### 1. 数据源调整
- ❌ 屏蔽 Tushare `moneyflow` 接口（资金流向数据）
- ❌ 屏蔽 Tushare `cyq_perf` 接口（筹码分布数据）
- ❌ 屏蔽 Tushare `stock_basic` 接口（股票列表）
- ✅ 完全使用 pytdx 数据源

#### 2. 功能替代方案

**资金流向分析**：
- 原方案：使用 Tushare 特大单/大单数据
- 新方案：基于 pytdx 成交量和价格变化推断
  - 量价配合分析（50 分）
  - 连续放量判断（30 分）
  - 成交量趋势分析（20 分）

**股票池获取**：
- 原方案：从 Tushare 获取完整股票列表
- 新方案：使用 pytdx 常见股票代码（200 只）
  - 深圳市场：000-099
  - 上海市场：600-699
  - 自动过滤股价<2 元的 ST 嫌疑股

**筹码分析**：
- 原方案：使用 Tushare 筹码分布数据
- 新方案：暂时使用默认评分（50 分）

#### 3. 权重调整

由于数据源变化，调整了综合评分权重：

```python
# 原权重
综合评分 = (
    资金流向 * 0.50 +    # Tushare 特大单/大单
    Alpha 因子 * 0.25 +   # 101 Alpha
    筹码分析 * 0.15 +     # Tushare 筹码
    动量分析 * 0.10       # 日内动量
)

# 新权重（当前版本）
综合评分 = (
    资金流向 * 0.40 +    # 成交量推断
    Alpha 因子 * 0.35 +   # 增加权重
    筹码分析 * 0.00 +     # 暂时屏蔽
    动量分析 * 0.25       # 增加权重
)
```

#### 5. 评分系统变化

**资金流向评分**（基于成交量推断）：

| 情况 | 评分 | 说明 |
|-----|------|------|
| 放量上涨（量比>1.5，涨幅>3%） | +50 | 最强信号 |
| 温和放量上涨（量比>1.2，涨幅>0） | +30 | 强势信号 |
| 连续 5 日放量 | +30 | 持续性 |
| 连续 3 日放量 | +20 | 持续性 |
| 成交量趋势向上（>1.3） | +20 | 趋势 |
| 放量下跌（量比>1.5，跌幅>0） | -40 | 危险信号 |
| 缩量下跌（量比<0.8） | -10 | 洗盘可能 |

**注**：当前评分基于量价关系推断，准确性低于真实资金流数据

## ⚠️ 当前版本限制

### 1. 数据准确性
- ❌ 无法获取真实特大单/大单数据
- ❌ 无法获取筹码分布数据
- ⚠️ 资金流向基于成交量推断（准确性中等）

### 2. 选股范围
- ⚠️ 仅覆盖 200 只常见股票
- ❌ 无法获取完整市场股票列表
- ⚠️ 可能遗漏部分强势股

### 3. 策略效果
- ⚠️ 预期胜率从 60% 降至 50-55%
- ⚠️ 需要更多手动筛选
- ⚠️ 建议轻仓测试

## 🐛 已知问题

### 1. 股票池较小
- **问题**：仅 200 只常见股票
- **影响**：可能遗漏强势股
- **解决**：等待 Tushare 恢复

### 2. ST 过滤不精确
- **问题**：仅通过股价<2 元简单判断
- **影响**：可能误判或漏判
- **解决**：手动检查或等待 Tushare 恢复

### 3. 资金流推断误差
- **问题**：成交量≠资金流向
- **影响**：评分准确性下降
- **解决**：结合其他指标综合判断
