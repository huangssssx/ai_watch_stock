## 二、核心函数与运算符定义（不区分大小写）
| 函数/运算符 | 作用与参数意义 |
|-------------|----------------|
| `abs(x)` | 取x的绝对值 |
| `log(x)` | 取x的自然对数 |
| `sign(x)` | 取x的符号：x>0返回1，x=0返回0，x<0返回-1 |
| `+、-、*、/` | 标准算术运算符 |
| `>、<、==、||` | 标准逻辑运算符 |
| `x ? y : z` | 三元条件运算符：若x为真则返回y，否则返回z |
| `rank(x)` | **横截面排名**：对同一时间点所有股票的x值进行排名，返回排名值 |
| `delay(x, d)` | 返回**d天前**的x值 |
| `correlation(x, y, d)` | 计算x和y**过去d天**的时间序列相关系数 |
| `covariance(x, y, d)` | 计算x和y**过去d天**的时间序列协方差 |
| `scale(x, a=1)` | 对x进行标准化，使得`sum(abs(x))=a`（默认a=1） |
| `delta(x, d)` | 计算x的d日差值：今日x值 - d天前的x值 |
| `signedpower(x, a)` | 计算x的a次幂，**保留x的符号** |
| `decay_linear(x, d)` | 过去d天的**线性衰减加权移动平均**，权重为d, d-1, ..., 1（权重和归一化为1） |
| `indneutralize(x, g)` | **行业中性化**：按组g（板块/行业/子行业等）对x进行横截面去均值 |
| `ts_{O}(x, d)` | 对x**过去d天**的时间序列应用算子O（非整数d自动向下取整） |
| `ts_min(x, d)` / `min(x, d)` | 返回x过去d天的最小值 |
| `ts_max(x, d)` / `max(x, d)` | 返回x过去d天的最大值 |
| `ts_argmax(x, d)` | 返回x过去d天中**最大值出现的天数**（相对于今日） |
| `ts_argmin(x, d)` | 返回x过去d天中**最小值出现的天数**（相对于今日） |
| `ts_rank(x, d)` | 返回x在过去d天**时间序列中的排名** |
| `sum(x, d)` | 返回x过去d天的总和 |
| `product(x, d)` | 返回x过去d天的乘积 |
| `stddev(x, d)` | 返回x过去d天的移动标准差 |

## 三、输入数据参数说明
| 参数 | 意义 |
|------|------|
| `returns` | 每日收盘价对收盘价的收益率 |
| `open` | 每日开盘价 |
| `close` | 每日收盘价 |
| `high` | 每日最高价 |
| `low` | 每日最低价 |
| `volume` | 每日成交量 |
| `vwap` | 每日**成交量加权平均价格** |
| `cap` | 股票市值 |
| `adv{d}` | 过去d天的**平均日美元成交量**（如`adv20`表示过去20天平均日美元成交量） |
| `IndClass` | 通用行业分类占位符（支持GICS、BICS、NAICS、SIC等），可指定层级为`sector`（板块）、`industry`（行业）、`subindustry`（子行业）等；同一Alpha中可使用不同的行业分类 |

## 四、关键说明
1.  **交易延迟**：4个**delay-0 Alpha**（#42、#48、#53、#54）需在计算当日收盘时交易；其余均为**delay-1 Alpha**，在计算次日开盘/盘中交易
2.  **信号类型**：约70%为均值回归信号，30%为动量信号；复杂Alpha通常混合两种逻辑
3.  **行业中性**：使用`indneutralize`的Alpha已消除行业整体波动影响，仅捕捉行业内个股的相对收益
4.  **参数优化**：部分Alpha使用非整数天数参数（如3.92795），是原始论文中优化后的结果，实际使用可向下取整
