## 🎯 策略核心逻辑

### 一、主力资金流向分析（40% 权重）

**数据来源**：
- ~~Tushare `moneyflow` 接口~~（暂时屏蔽）
- pytdx 成交量分析（当前使用）

**核心指标**（基于成交量推断）：
1. **量价配合**（权重 50%）
   - 放量上涨（量比>1.5，涨幅>3%）：+50 分
   - 温和放量上涨：+30 分
   - 放量下跌：-40 分
   - 缩量下跌：-10 分

2. **连续性**（权重 30%）
   - 连续 5 日放量：+30 分
   - 连续 3 日放量：+20 分

3. **成交量趋势**（权重 20%）
   - 5 日均量趋势向上（>1.3）：+20 分
   - 趋势向上（>1.0）：+10 分

**注**：Tushare 接口恢复后将使用特大单/大单数据

### 二、101 Alpha 因子（25% 权重）

**核心 Alpha**：

```python
# Alpha#2: 量价相关性
alpha = -1 * correlation(
    rank(delta(log(volume), 2)), 
    rank((close - open) / open), 
    6
)

# Alpha#6: 开盘量价关系
alpha = -1 * correlation(open, volume, 10)

# Alpha#12: 量价方向
alpha = sign(delta(volume, 1)) * (-1 * delta(close, 1))
# 解读：放量下跌=主力卖出，缩量下跌=洗盘

# Alpha#42: VWAP 相对位置（强烈推荐）
alpha = rank(vwap - close) / rank(vwap + close)
# 解读：收盘>VWAP 表示主力控盘

# Alpha#101: 日内动量
alpha = (close - open) / (high - low + 0.001)
# 解读：>0.7 表示强势
```

### 四、动量分析（10% 权重）

**核心逻辑**：

- 涨幅 >7%：40 分（可能涨停）
- 涨幅 <1%：30 分

## 🔍 主力识别技巧

### 真实主力信号（按重要性排序）

1. **特大单持续净流入** ⭐⭐⭐⭐⭐
   - 最真实的主力信号
   - 难以伪装（需要真金白银）
   - 连续 3 日以上更可靠

2. **尾盘放量拉升** ⭐⭐⭐⭐⭐
   - 主力急于建仓/抢筹
   - 最后 30 分钟成交额占比>25%
   - 尾盘大单集中

3. **成交量异常放大** ⭐⭐⭐⭐
   - Z-Score > 2
   - 量比 > 1.5
   - 连续放量

4. **VWAP 偏离** ⭐⭐⭐⭐
   - 收盘价持续高于 VWAP
   - 主力控盘迹象

5. **筹码集中度提升** ⭐⭐⭐
   - 平均成本收敛
   - 胜率提升

### 警惕信号

❌ **放量下跌**：主力出货
- 量比 > 1.5 且跌幅 > 3%

❌ **缩量上涨**：可疑
- 量比 < 0.8 且涨幅 > 5%

❌ **尾盘跳水**：主力出逃
- 最后 30 分钟跌幅 > 2%

❌ **散户主导**：
- 小单净流入 > 主力净流入

## 💡 实战应用

### 选股流程

```
1. 14:25-14:30  准备阶段
   - 检查市场状态
   - 准备股票池

2. 14:30-14:40  初筛阶段
   - 过滤 ST、科创板
   - 筛选股价/市值范围
   - 筛选量比 > 1.5

3. 14:40-14:50  评分阶段
   - 计算资金流向评分
   - 计算 Alpha 评分
   - 计算筹码评分
   - 计算动量评分

4. 14:50-15:00  决策阶段
   - 排序选股（Top 5）
   - 快速下单
```

### 仓位管理

**建议**：
- 单票仓位 ≤ 10%
- 总仓位 ≤ 30%
- 持有 1-3 天

### 止盈止损

**止盈**：
- 隔日高开 > 3%：考虑止盈
- 隔日涨停：持有
- 隔日冲高回落：果断止盈

**止损**：
- 跌破买入价 -3%：止损
- 跌破 5 日线：止损
- 主力转向（特大单流出）：止损

## 📞 常见问题

### Q1: 为什么选的股票第二天跌了？

A: 没有任何策略 100% 胜率。关键是：
- 严格执行止损
- 长期统计胜率>55% 即可盈利
- 检查是否偏离策略（如追高）

### Q2: 如何优化策略？

A: 建议方向：
1. 增加更多 Alpha 因子
2. 优化权重配置
3. 加入大盘择时
4. 结合板块轮动

### Q3: 可以盘中实时选股吗？

A: 可以，但建议：
- 使用 1 分钟 K 线
- 尾盘 14:30 后执行
- 避免早盘冲动交易

### Q4: 资金流向数据准确吗？

A: 
- **当前版本**：使用成交量推断，准确性中等
- **Tushare 恢复后**：使用特大单/大单数据，准确性高
  - 特大单最准确（机构/游资）
  - 大单次之
  - 中单/小单参考即可
