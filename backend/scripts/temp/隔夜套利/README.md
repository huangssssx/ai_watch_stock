# 隔夜套利策略 - 完整思路与实现逻辑

## 📋 策略概述

**策略类型**：短线套利（持有 1 天）

**核心逻辑**：利用市场情绪惯性，在尾盘买入强势股，次日早盘卖出

**执行时间**：
- 买入：14:30-14:50（尾盘）
- 卖出：次日 09:25-10:00（早盘）

**预期收益**：
- 胜率：55-65%
- 平均收益：1-3%/次
- 月收益：10-20%（复利）

---

## 🎯 策略核心思想

### 一、为什么能做隔夜套利？

1. **情绪惯性**
   - 强势股的情绪会延续到次日
   - 涨停股次日高开概率大
   - 板块热点会持续 1-3 天

2. **T+1 制度红利**
   - 当日买入，次日可卖
   - 利用早盘冲高获利了结

3. **尾盘确定性高**
   - 14:30 后当日走势基本确定
   - 主力意图暴露（不演戏了）
   - 避免盘中波动风险

### 二、策略本质

```
隔夜套利 = 情绪判断 + 选股 + 纪律执行

情绪判断（60%）：决定今天能不能做
选股（30%）：决定买什么
纪律执行（10%）：决定能不能持续盈利
```

## 📊 策略实现逻辑

### 第一步：情绪过滤（三步法）⭐⭐⭐⭐⭐

**目的**：判断今天是否适合做隔夜套利

**核心问题**：
- 如果市场情绪差，强势股次日可能直接跌停
- 情绪好时，胜率从 50% 提升到 65%+

#### 1.1 晋级率检查（权重 40%）

**计算方式**：
```python
晋级率 = 昨日涨停股今日高开或继续涨停数量 / 昨日涨停总数 * 100%
```

**判断标准**：
- ≥ 60%：情绪好 ✅
- 40-60%：情绪一般 ⚠️
- < 40%：情绪差 ❌

**为什么重要**：
- 晋级率反映接力资金意愿
- 高晋级率 = 资金愿意追高
- 低晋级率 = 核按钮（跌停）多

**数据来源**：
- Tushare `limit_list_d`：获取昨日涨停股列表
- Tushare `daily`：获取今日行情
- pytdx：实时行情判断是否高开

#### 1.2 跌停预警检查（权重 40%）

**检查内容**：
1. 昨日核心龙头股今日是否跌停
2. 昨日最高板今日表现
3. 昨日一字板今日表现

**判断标准**：
- 核心股跌停数 = 0：安全 ✅
- 核心股跌停数 = 1-2：警惕 ⚠️
- 核心股跌停数 ≥ 3：危险 ❌

**核心股定义**：
- 昨日连板股（2 板以上）
- 昨日板块龙头
- 昨日一字涨停股

**为什么重要**：
- 核心股跌停 = 情绪崩盘信号
- 龙头核按钮会传染
- 次日可能集体低开

**数据来源**：
- Tushare `limit_list_d`：获取昨日连板股
- Tushare `stk_limit`：获取涨跌停价格
- pytdx：实时判断是否跌停

#### 1.3 梯队完整性检查（权重 20%）

**检查内容**：
1. 昨日最强板块的龙头股今日表现
2. 中军股（大市值）今日表现
3. 跟风股今日表现

**梯队完整性判断**：
- 完整：龙头 + 中军 + 跟风都在涨 ✅
- 一般：龙头涨，中军震荡 ⚠️
- 差：龙头跌或板块集体回调 ❌

**为什么重要**：
- 完整梯队 = 板块行情持续
- 龙头独木难支 = 行情难持续
- 板块效应是强势股保障

**数据来源**：
- Tushare `concept`：板块数据
- Tushare `daily`：个股行情

### 第二步：个股筛选（过滤器）⭐⭐⭐⭐

**目的**：从强势股中选出最可能次日高开的股票

**执行时间**：14:20-14:30 确认，14:30-14:50 买入

#### 2.1 选股条件（必须同时满足）

**条件 1：当日最强板块内的涨停股**

```python
# 逻辑
最强板块 = 板块内涨停股数量最多的板块
板块效应 = 资金集中攻击方向
次日溢价 = 板块持续性
```

**条件 2：封板时间 < 14:30**

```python
# 为什么重要
早盘封板 = 主力态度坚决
午后封板 = 跟风或犹豫
尾盘封板 = 偷袭，次日易低开

封板时间判断：
- 10:00 前：强势 ✅
- 10:00-14:00：中等 ⚠️
- 14:00-14:30：较弱 ⚠️
- 14:30 后：放弃 ❌
```

**条件 3：封单金额 ≥ 3000 万**

```python
# 计算方式
封单金额 = 买一价 * 买一量

为什么重要：
- 封单大 = 主力真金白银
- 封单小 = 可能炸板
- 封单变化 = 主力态度

判断标准：
- ≥ 1 亿：非常强 ✅
- 5000 万 -1 亿：强 ✅
- 3000 万 -5000 万：中等 ⚠️
- < 3000 万：弱 ❌
```

**条件 4：板块内最早涨停的前 3 名**

```python
# 逻辑
龙头 = 最早涨停 + 带动性
龙二 = 紧随其后
龙三 = 板块确认

为什么只买前三：
- 龙头溢价最高
- 跟风股次日易低开
- 资金聚焦效应
```

**条件 5：非尾盘偷袭板（无反复炸板）**

```python
# 炸板判断
炸板 = 涨停后打开
反复炸板 = 打开≥2 次

为什么重要：
- 反复炸板 = 分歧大
- 尾盘偷袭 = 主力不自信
- 次日易低开
```

**条件 6：当日跌停数 < 20 只**

```python
# 市场情绪指标
跌停数 ≥ 20：情绪差，放弃
跌停数 10-20：谨慎
跌停数 < 10：安全

为什么重要：
- 跌停多 = 恐慌情绪
- 恐慌会传染
- 次日集体低开
```

**条件 7：大盘 14:00 后走势平稳**

```python
# 大盘判断
大盘 = 上证指数/深证成指

走势判断：
- 14:00 后上涨：安全 ✅
- 14:00 后震荡：可接受 ⚠️
- 14:00 后跳水：放弃 ❌

为什么重要：
- 大盘影响个股情绪
- 尾盘跳水 = 恐慌
- 覆巢之下无完卵
```

#### 2.2 选股流程

```
14:20 开始筛选
   ↓
1. 获取当日涨停股列表
   ↓
2. 筛选封板时间 < 14:20
   ↓
3. 筛选封单金额 ≥ 3000 万
   ↓
4. 按板块分类，找最强板块
   ↓
5. 筛选最强板块的前 3 只
   ↓
6. 检查是否有反复炸板
   ↓
7. 检查大盘走势
   ↓
14:30 返回最终名单（最多 3 只）
```

### 第三步：卖出策略（纪律执行）⭐⭐⭐⭐⭐

**目的**：确保盈利，控制亏损

**执行时间**：次日 09:25-10:00

**核心原则**：
- 不贪婪（不格局）
- 不幻想（破位就走）
- 不拖延（10:00 前必须清仓）

#### 3.1 卖出预案

**情况 1：高开 ≥ 5%**

```python
操作：
1. 09:25 集合竞价后卖半仓
2. 剩余仓位观察
   - 继续冲高：持有到 09:35
   - 掉头向下：立即清仓
   - 翻绿：立即清仓

逻辑：
- 高开 5% 已获利，先锁定一半利润
- 防止冲高回落
- 避免由盈转亏
```

**情况 2：高开 2-5%**

```python
操作：
1. 09:25 开盘全卖

逻辑：
- 符合预期，落袋为安
- 不博更高收益
- 避免意外
```

**情况 3：平开 ±2%**

```python
操作：
1. 观察 3 分钟（09:25-09:28）
2. 不翻红（不涨）立即全出
3. 翻红后回落也立即出

逻辑：
- 平开低于预期
- 3 分钟不涨 = 弱
- 快进快出
```

**情况 4：低开**

```python
操作：
1. 09:25 挂跌停价全出

逻辑：
- 低开 = 失败
- 立即止损，不抱幻想
- 防止深套

为什么挂跌停价：
- 确保成交
- 实际成交价是开盘价
- 避免卖不出去
```

#### 3.2 特殊情况处理

**情况 A：直接涨停**

```python
操作：
- 持有
- 次日继续按预案执行

逻辑：
- 超预期，享受溢价
- 但次日仍按纪律执行
```

**情况 B：直接跌停**

```python
操作：
- 认亏
- 次日开盘立即清仓

逻辑：
- 承认失败
- 防止继续下跌
- 留得青山在
```

**情况 C：隔夜美股大跌**

```python
判断：
- 美股跌幅 > 2%：直接挂跌停清仓
- 美股跌幅 1-2%：按预案执行
- 美股上涨：正常执行

逻辑：
- 美股影响 A 股情绪
- 大跌会低开
- 先出来观望
```

## 💡 策略核心要点

### 一、情绪是核心（60% 重要性）

**为什么情绪最重要**：
1. 情绪好 = 容错率高
   - 即使选错股也可能赚钱
2. 情绪差 = 龙头也跌停
   - 选股再好也亏钱
3. 情绪决定胜率
   - 好情绪：胜率 65%
   - 差情绪：胜率 40%

**情绪判断三要素**：
```
晋级率（40%）：最直接的情绪指标
跌停预警（40%）：风险信号
梯队完整（20%）：持续性判断
```

### 二、选股要聚焦（30% 重要性）

**聚焦原则**：
1. 只买龙头（前 3 名）
2. 只买最强板块
3. 只买早盘封板

**为什么聚焦**：
- 资金有限，买太多分散
- 龙头溢价最高
- 跟风股风险大

### 三、纪律是关键（10% 重要性但决定生死）

**三大纪律**：
1. 次日必须卖出（不格局）
2. 10:00 前清仓（不拖延）
3. 低开立即止损（不幻想）

**为什么纪律重要**：
- 隔夜套利是套利，不是投资
- 贪婪会由盈转亏
- 幻想会深套

## ⚠️ 风险控制

### 一、仓位管理

**标准仓位**：
- 单票：≤ 10%
- 总仓：≤ 30%

**轻仓情况**：
- 情绪两步通过：单票≤5%
- 市场震荡：单票≤5%

**空仓情况**：
- 情绪一步通过或全不通过
- 美股大跌（>2%）
- 当日跌停数 > 20

### 二、止损纪律

**硬性止损**：
- 低开 > -2%：立即止损
- 次日跌停：认亏清仓

**软性止损**：
- 平开不涨：3 分钟不出红就走
- 冲高回落：翻绿立即走

### 三、特殊情况

**必须空仓的情况**：
1. 情绪判断 ≤ 1 步通过
2. 当日跌停数 > 20
3. 美股大跌 > 2%
4. 大盘尾盘跳水
5. 连续亏损 3 次

**必须轻仓的情况**：
1. 情绪判断 2 步通过
2. 封单金额 < 5000 万
3. 板块梯队一般
4. 刚连续亏损

### 核心算法

**1. 晋级率计算**：
```python
def calculate_promotion_rate(yesterday_limits, today_data):
    """
    计算晋级率
    yesterday_limits: 昨日涨停股列表
    today_data: 今日行情数据
    """
    total = len(yesterday_limits)
    promoted = 0
    
    for stock in yesterday_limits:
        today_close = today_data[today_data['ts_code'] == stock]['close'].iloc[0]
        yesterday_close = stock['close']
        
        # 高开或继续涨停
        if today_close > yesterday_close * 1.02:  # 高开 2% 以上
            promoted += 1
    
    return promoted / total if total > 0 else 0
```

**2. 封单金额计算**：
```python
def calculate_seal_order(realtime_data):
    """
    计算封单金额
    realtime_data: 实时行情
    """
    bid1_price = realtime_data['bid1']
    bid1_vol = realtime_data['bid_vol1']
    
    seal_amount = bid1_price * bid1_vol
    
    return seal_amount
```

**3. 封板时间判断**：
```python
def get_limit_time(k_data, limit_price):
    """
    获取封板时间
    k_data: 分钟 K 线
    limit_price: 涨停价
    """
    for i, bar in enumerate(k_data):
        if bar['close'] >= limit_price * 0.99:  # 接近涨停
            # 检查是否封住
            if i + 5 < len(k_data):
                next_bars = k_data[i:i+5]
                if all(bar['close'] >= limit_price * 0.99 for bar in next_bars):
                    return bar['datetime']
    
    return None
```

## 📞 常见问题

### Q1: 为什么限定 14:30 执行？

A: 
- 14:30 后当日走势确定
- 主力意图暴露（不演戏）
- 避免盘中波动
- 太早选可能炸板

### Q2: 为什么只买前 3 名？

A:
- 龙头溢价最高
- 资金聚焦效应
- 跟风股次日易低开
- 实证数据显示龙一龙二胜率最高

### Q3: 次日必须卖出吗？

A:
