# 筹码扫描脚本模块导入修复

**时间**: 2026-03-27
**任务**: 修复 ModuleNotFoundError: No module named 'backend'

## 问题分析

脚本路径: `backend/scripts/筹码/全市场筹码聚集度扫描.py`

`_project_root()` 函数原实现只往上跳两级目录：
```python
return os.path.abspath(os.path.join(here, "..", ".."))
```

但脚本位于 `backend/scripts/筹码/` 深层目录，需要跳**三级**才能到达项目根目录。

## 修复方案

修改 `_project_root()` 函数，往上跳三级：
```python
return os.path.abspath(os.path.join(here, "..", "..", ".."))
```

## 验证结果

```bash
cd backend && python3 scripts/筹码/全市场筹码聚集度扫描.py --max-stocks 10
```

...
