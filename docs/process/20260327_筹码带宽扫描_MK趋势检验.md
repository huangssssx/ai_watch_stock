# 执行记录：筹码带宽扫描脚本 + Mann-Kendall 趋势检验

**日期**: 2026-03-27
**脚本**: backend/scripts/筹码/全市场筹码聚集度扫描.py

## 功能实现

1. **筹码带宽计算**: `(cost_95pct - cost_5pct) / weight_avg`
2. **T日带宽范围筛选**: 10%~20% (默认)
3. **Mann-Kendall 趋势检验**: 使用 `scipy.stats.kendalltau`
4. **趋势筛选**: 最近20个交易日带宽呈下降趋势

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| --min-band | 0.10 | 最小带宽 |
| --max-band | 0.20 | 最大带宽 |
| --trend-lookback | 20 | 趋势回溯天数 |
| --trend-min-tau | -0.3 | 最小tau值(负=下降) |
| --trend-max-pvalue | 0.05 | 最大p值(显著性) |
| --no-trend-filter | false | 禁用趋势筛选 |

## 缓存

- 文件: `backend/data/chip_band_cache.csv`

## 输出字段

- ts_code, name, chip_band, close
- cost_5pct, cost_95pct, weight_avg, winner_rate
- trend_tau, trend_pvalue, trend_change_rate
