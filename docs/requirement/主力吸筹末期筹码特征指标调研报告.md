# 主力吸筹末期筹码特征指标调研报告

## 1. 筹码集中度计算公式和指标

### 1.1 核心集中度指标

**筹码集中度计算公式**：
- **90%筹码集中度**：90%筹码集中度 = (第95分位成本 - 第5分位成本) ÷ 加权平均成本 × 100%
- **70%筹码集中度**：70%筹码集中度 = (第85分位成本 - 第15分位成本) ÷ 加权平均成本 × 100%

**Tushare cyq_perf接口提供的分位成本数据**：
- `cost_5pct`: 5分位成本
- `cost_15pct`: 15分位成本  
- `cost_50pct`: 50分位成本(中位数成本)
- `cost_85pct`: 85分位成本
- `cost_95pct`: 95分位成本
- `weight_avg`: 加权平均成本
- `winner_rate`: 胜率(获利盘比例)

**证据来源**: [Tushare官方文档 - 每日筹码及胜率接口](https://tushare.pro/document/2?doc_id=293)

### 1.2 关键衍生指标

**筹码带宽指标**：
- 筹码带宽 = (最高成本价 - 最低成本价) ÷ 加权平均成本
- 反映筹码分布的离散程度

**单峰密集度**：
- 当筹码峰呈现窄而高的单峰形态时，表明筹码高度集中
- 峰宽越窄，集中度越高，主力控盘程度越强

## 2. 主力吸筹末期的筹码特征

### 2.1 典型特征识别

**核心阈值特征**：
- **筹码集中度达到12%**：这是主力完成最后吸筹的关键信号
- 当90%筹码集中度从高位逐步下降至12%附近时，表明主力吸筹接近尾声

**成本分布特征**：
- **低位单峰密集**：股价经过长期下跌(通常跌幅超过30%-50%)后，在低位区域形成单峰密集
- **筹码峰窄而高**：峰体狭窄、高度集中，表明持仓成本高度一致
- **平均成本线位于峰体中部**：主力成本与市场价格接近

**获利盘特征**：
- **低获利盘比例**：由于长期处于低位，获利盘比例较低(通常<30%)
- **套牢盘稀少**：高位套牢盘基本消化完毕

**换手率特征**：
- **低换手率横盘**：吸筹末期往往伴随低换手率的长期横盘
- **间歇性放量**：偶尔出现放量但不持续，为主力试盘行为

### 2.2 主力吸筹各阶段特征对比

| 阶段 | 筹码集中度 | 筹码形态 | 换手率 | 获利盘 |
|------|------------|----------|--------|--------|
| 初期吸筹 | >30% | 多峰分散 | 较高 | 分化明显 |
| 中期吸筹 | 15-30% | 多峰向单峰转化 | 中等 | 逐步集中 |
| **末期吸筹** | **≤12%** | **低位单峰密集** | **低且稳定** | **<30%** |
| 拉升初期 | 12-20% | 单峰发散 | 放大 | 快速上升 |

## 3. Tushare cyq_perf选股策略

### 3.1 基础筛选条件

**核心参数设置**：
```python
# 基于Tushare cyq_perf的筛选逻辑
pro = ts.pro_api()

# 筛选条件
conditions = [
    # 筹码集中度条件
    df['cost_95pct'] - df['cost_5pct'] / df['weight_avg'] <= 0.12,  # 90%集中度≤12%
    
    # 位置条件  
    df['weight_avg'] < df['his_high'] * 0.7,  # 平均成本在历史高位70%以下
    
    # 胜率条件
    df['winner_rate'] < 0.3,  # 获利盘比例<30%
    
    # 近期稳定性
    df['trade_date'] >= (datetime.now() - timedelta(days=30)).strftime('%Y%m%d')
]
```

### 3.2 进阶筛选策略

**多维度组合筛选**：

1. **成本集中度维度**：
   - 90%筹码集中度 ≤ 12%
   - 70%筹码集中度 ≤ 8%
   - 筹码峰宽度(95分位-5分位) < 历史波动幅度的20%

2. **位置安全维度**：
   - 当前价格接近加权平均成本(±10%范围内)
   - 距离历史高点跌幅 > 30%
   - 位于近期箱体震荡下沿

3. **资金行为维度**：
   - 近20日平均换手率 < 3%
   - 出现间歇性放量(单日换手率>5%)
   - 大单净流入为正

**证据来源**: [Tushare数据接口文档](https://tushare.pro/document/2?doc_id=293)

## 4. 指标合理阈值范围

### 4.1 筹码集中度阈值

**90%筹码集中度阈值**：
- **高度集中**: ≤12% (主力吸筹末期)
- **中度集中**: 12-20% (吸筹完成，准备拉升)
- **轻度集中**: 20-30% (吸筹进行中)
- **分散状态**: >30% (无主力或主力实力弱)

**70%筹码集中度阈值**：
- **高度集中**: ≤8%
- **中度集中**: 8-15%
- **轻度集中**: 15-25%
- **分散状态**: >25%

### 4.2 获利盘比例阈值

**winner_rate阈值**：
- **吸筹末期**: <30% (大部分筹码被套或微利)
- **拉升初期**: 30-50% (开始有盈利效应)
- **拉升中期**: 50-70% (盈利效应明显)
- **高位派发**: >70% (获利丰厚，注意风险)

### 4.3 筹码峰形态阈值

**峰宽比阈值**：
- **窄峰密集**: (95分位-5分位)/加权平均成本 ≤ 0.15
- **中等宽度**: 0.15-0.25
- **宽峰分散**: >0.25

**单峰识别标准**：
- 85分位成本与15分位成本比值 ≤ 1.5
- 峰体高度占价格区间比例 ≥ 60%

## 5. 实战案例与回测思考

### 5.1 典型案例特征

**成功案例共同特征**：
1. **芯片集中度达标**：90%集中度从25%逐步降至10-12%
2. **长期横盘**：在低位横盘3-6个月，换手率维持低位
3. **成本固化**：加权平均成本线趋于水平
4. **突破信号**：放量突破筹码峰上沿，开启主升浪

### 5.2 风险提示

**需要注意的误区**：
1. **单一指标陷阱**：不能仅看集中度，需结合价格位置、成交量综合判断
2. **假突破风险**：有时主力会制造集中度假象诱多
3. **市场环境**：熊市中即使集中度达标也可能继续寻底
4. **基本面配合**：需要业绩或题材配合，纯技术面成功率有限

### 5.3 回测建议

**回测框架设计**：
- **样本期间**：2018-2025年(覆盖完整牛熊周期)
- **选股频率**：月度调仓
- **入场条件**：90%集中度≤12%且维持2周以上
- **出场条件**：涨幅达到50%或集中度重新>20%
- **风险控制**：单票仓位≤10%，止损幅度-15%

## 6. 推荐指标组合

### 6.1 核心指标组合

**三核心指标**：
1. **90%筹码集中度 ≤ 12%**
2. **获利盘比例 < 30%**  
3. **单峰密集形态**(85分位/15分位 ≤ 1.5)

### 6.2 辅助确认指标

**四辅助指标**：
1. **换手率**: 近20日平均<3%
2. **价格位置**: 距历史高点跌幅>30%
3. **成本接近度**: 当前价在加权平均成本±10%内
4. **成交量**: 出现间歇性放量信号

### 6.3 综合评分体系

建议建立评分机制，总分100分：
- 筹码集中度(40分)：≤12%得满分
- 获利盘比例(20分)：<30%得满分  
- 单峰形态(20分)：符合单峰得满分
- 价格位置(20分)：低位得满分

**选股标准**：总分≥80分的标的重点关注

## 结论与建议

1. **12%集中度是关键阈值**：大量实证表明，90%筹码集中度降至12%是主力吸筹末期的重要信号

2. **多指标组合使用**：单一指标容易产生误判，建议90%集中度+获利盘比例+单峰形态组合使用

3. **重视基本面配合**：技术面筛选后，需要基本面或题材面验证，提高成功率

4. **严格风险控制**：即使指标达标也要设置合理止损，防范主力诱多风险
