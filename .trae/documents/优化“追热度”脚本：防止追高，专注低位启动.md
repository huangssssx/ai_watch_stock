# 策略优化计划：寻找“低位潜伏”股

为了避免“追高被套”，我们将调整筛选逻辑，核心从“追涨”转变为“潜伏”。利用 API 中包含的 `60日涨跌幅` 数据，我们可以有效识别股票所处的位置。

## 1. 筛选逻辑调整

### 新增“位置”过滤（核心）
*   **60日涨跌幅**：限制在 `< 20%`。
    *   **目的**：剔除过去两个月已经大幅炒作过的股票，确保股价处于相对低位或横盘整理区。
*   **当日涨跌幅**：收窄至 `1.0%` ~ `5.0%`。
    *   **目的**：寻找刚刚启动、温和上涨的股票，避开已经暴涨（>5%）或涨停的票。

### 强化“资金”信号
*   **量比**：保持 `> 1.5`。
    *   **解读**：低位放量是主力吸筹或启动的最重要信号（量在价先）。
*   **换手率**：`3%` ~ `12%`。
    *   **解读**：保持适度活跃，但不要过热（>15% 通常意味着高潮或出货）。

## 2. 综合打分公式优化
我们将调整评分权重，让“低位”和“放量”成为得分关键：

*   **量比 (资金强度)**：权重 **40%** (不变，核心指标)
*   **位置优势 (60日涨幅)**：权重 **30%** (新指标，涨幅越小得分越高)
*   **活跃度 (换手率)**：权重 **30%**

## 3. 预期效果
*   **形态**：筛选出的股票多为“底部横盘突破”或“上升通道初期的补涨股”。
*   **安全性**：由于剔除了近期大涨股，即便回调，下方支撑较强，避免“半山腰”接盘。
*   **点评更新**：AI 点评将增加“低位补涨”、“底部放量”等描述，辅助决策。
